\# 🚀 Pipeline de Vendas - CyNext



Dashboard interativo para captura e visualização de leads em tempo real.



\## 🔗 Demo ao vivo

👉 https://cynext-dashboard-cyd5zhez7lpxua4itrrqs2.streamlit.app



\## 🛠️ Tecnologias utilizadas



| Ferramenta | Função |

|---|---|

| n8n | Automação e captura de leads via Webhook |

| Supabase | Banco de dados PostgreSQL na nuvem |

| Python + Streamlit | Dashboard interativo em tempo real |

| GitHub | Versionamento e deploy automático |



\## ⚙️ Como funciona



Lead enviado via Webhook → n8n recebe e processa → Supabase armazena → Dashboard exibe em tempo real



\## 📊 Funcionalidades



\- Total de leads capturados

\- Taxa de conversão

\- Custo médio por lead

\- Gráfico de leads por origem

\- Status do pipeline de vendas

\- Leads por dia e por cidade

\- Filtros por origem e status
